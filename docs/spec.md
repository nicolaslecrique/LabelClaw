# Product Spec

## Purpose

LabelClaw is a dataset labeling app where the user can create their own labeling UI.

## Business Specifications

The app is composed of one "configuration" tab and one "labelling" tab.

### Configuration tab

The configuration tab exposes a left panel with 4 rows: 3 text fields and a "generate labelling panel" button.

* "sample JSON Schema": a text field where the user writes the labeling task input schema
* "label JSON Schema": a text field where the user writes the labeling task output schema
* "UI prompt": a text field where the user describes the React component that will be generated by the LLM through the backend and rendered dynamically

When the user clicks the button, a React component is generated and rendered on the right panel with sample data.

The generated React component takes a data instance matching the sample JSON Schema and exposes a function that returns an output matching the label JSON Schema.

* On top of the right panel, there is a "save labelling panel" button that saves the generated code in a backend-managed configuration file
